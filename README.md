# react-emaily
Simple email survey with sendgrid api 

npm install

create dev.js inside client/config with:
module.exports = {
  googleClientID: ,
  googleClientSecret: ,
  mongoUri: ,
  cookieKey: ,
  stripePublishableKey: ,
  stripeSecretKey: ,
  sendGridKey: ,
  redirectDomain: ,
};

and npm run dev at the server path to start
